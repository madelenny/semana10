Backlog de la solución comunitaria

Prioridad 1.
Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

Prioridad 2.
Como coordinador quiero recibir una alerta en Telegram para revisar un nuevo reporte.

Prioridad 3.
Como miembro del equipo quiero consultar los reportes para hacer seguimiento.

Prioridad 4.
Como administrador quiero actualizar el estado de cada reporte para informar el avance.

Prioridad 5.
Como ciudadano quiero conocer el estado de mi reporte para saber si fue atendido.
